# DevTinder Frontend

##### Backend - https://github.com/virajpwdn/devTinder

---

The user-facing frontend for the DevTinder platform — a modern, responsive web application connecting developers through intelligent matchmaking, real-time chat, and seamless payments.

This project is built with production-grade standards, emphasizing:

- **Performance & Optimization** – Fast load times and smooth interactions
- **Scalability** – Component-based architecture for easy expansion
- **User Experience** – Intuitive UI with real-time features
- **State Management** – Centralized, predictable application state
- **Responsive Design** – Works flawlessly across all devices

---

## 🛠 Tech Stack

### Frontend Framework & Build Tools

- **React** – Component-based UI library
- **Vite** – Lightning-fast build tool and dev server

### State Management

- **Redux Toolkit** – Predictable state management with minimal boilerplate
- **Redux DevTools** – Time-travel debugging and state inspection

### Styling & UI Components

- **Tailwind CSS** – Utility-first CSS framework for rapid UI development
- **Daisy UI** – Pre-built, accessible React components on top of Tailwind

### Real-Time Communication

- **Socket.IO Client** – Real-time bidirectional communication for instant messaging

### API & Data Fetching

- **Axios** – Promise-based HTTP client for API calls
- **React Query** (Optional) – Data fetching and caching

### Development Tools

- **Prettier** – Code formatting
- **Vite Dev Server** – Fast HMR (Hot Module Replacement)

---

## ✨ Key Features

### 🔐 Authentication & User Management

- Secure user signup and login flows
- User profile creation and editing
- Persistent authentication state

### 💬 Real-Time Chat

- Instant one-to-one messaging with Socket.IO
- Live typing indicators
- Message history and conversation management
- Low-latency, responsive chat experience

### ❤️ Smart Matchmaking

- Browse potential developer matches
- Like, pass, or skip profiles
- Match notifications and instant connections
- Intuitive matching interface

### 💳 Payment Integration

- Seamless payment checkout flow
- Razorpay integration for secure transactions

### 📱 Responsive Design

- Mobile-first approach
- Adaptive layouts for all screen sizes
- Optimized touch interactions

### 🎨 Modern UI/UX

- Clean, intuitive interface
- Smooth animations and transitions
- Accessibility-first components (via Daisy UI)
